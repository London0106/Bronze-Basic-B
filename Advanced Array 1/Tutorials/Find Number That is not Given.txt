문제
Among the numbers from 1 to N, N−1 numbers are given without duplication. 
In this case, print the number from 1 to N that is not given.

입력
The first line is given as an integer N.
(5≤N≤1,000)

From the second line to the N-th line, integers are given, one per line.

출력
Print the number from 1 to N that is not given.

2
